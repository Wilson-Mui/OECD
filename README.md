There are three different visualizations of a similar underlying dataset, the OECD data on debt vs. GDP by country across turn of the millennium. The top bar chart focuses on the debt to GDP ratio for one year (2015), which sacrifices a time changing component of the data but is better able to highlight one group of data - that is, which countries had the most extreme ratios. We see that very clear it is Italy, Greece, and Japan.</p>

The second graph is more invovled, and incorporates the time element. What this graph emphasizes is how the ratio of debt vs. GDP changes through time for each country, with larger circles representing a larger ratio. This graph also facilitates comparison across countries at any given point in time. This graph shows a lot of information in a relatively concise manner, by incorporating the circle size as a visual cue.</p>

However, circle size does not convey precise numerical information well. A box plot is a better choice for showing this type of information. Additionally, I chose a box plot because it emphasizes a different component of the data. With this box plot the emphasis is on the range of ratio values, for example if we wanted to highlight the extreme ranges of fiscal policies per country (where there was a lot of spending and then a period of austerity). This graph was generated using the RAW tool, which limits some control. In this example the color is used to emphasize large ranges. With greater control I would also remove the numbers next to the box plots, they clutter the image, and I would re-order the countries with the largest ranges first (I sorted the data by range, but the RAW tool sorted by name, regardless)
